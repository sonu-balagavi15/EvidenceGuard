About Evidence Guard

Digital Evidence Management System

Evidence Guard is a Flask-based web application developed to securely manage digital evidence.

Features
  • User Registration & Login
  • Evidence Upload
  • SHA-256 Hash Generation
  • Evidence Verification
  • Chain of Custody
  • Evidence Search
  • Evidence History
  • Evidence Preview
  • Evidence Details
  • PDF Report Generation
  • Edit Evidence
  • Delete Evidence
Technology Stack
  • Python
  • Flask
  • SQLite
  • SQLAlchemy
  • Bootstrap 5
  • HTML / CSS
